# [Why Hooks Exist](https://reactjs.org/docs/hooks-intro.html#motivation) 💫

<ul>
  <Appear>
    <li>Hard to reuse stateful logic between components</li>
    <li>Complex components become hard to understand</li>
    <li>Classes confuse both people and machines</li>
  </Appear>
</ul>
